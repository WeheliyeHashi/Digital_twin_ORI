# Digital Twin Project for Oribiotech

This repository contains code for a digital twin project developed for Oribiotech. It includes scripts for training models, running inference, and organizing data.

## Files

- **Train.py**: Contains the training code for the digital twin model.
- **Inference.py**: Contains the inference code to run predictions using the trained model.
- **Organise_data.py**: Contains code to organize and preprocess the data used in the project.

## Usage

Please refer to each Python file for specific instructions on how to run them. Ensure you have the necessary dependencies installed before executing the scripts.
